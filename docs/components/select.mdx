# Option Object
The option object can now contain any value, this is because we have 2 methods
that allow us to derive an id and value

* `idFromValue: (value: T) => string | number`
* `labelFromValue: (value: T) => string | JSX.Element`

# value
A value should be an instance of `T`


# Select Component

```jsx
function Control ({ selectedValue, ...otherProps }) {

}

}

<SelectedComponent
  value={/* IOption */}
  options={/* IOption[] */}
  onChange={/* IOption || undefined */}
  components={{
    Control,
    Option,
  }}
/>
```

## Props
`onChange`: is called with the complete selected item (not just the value)
